GameController

Controlador de la vista del juego. Maneja la interfaz (mostrar cartas, turno, jugar carta, turnos de los bots).

package com.cincuentazo.controller;

public class GameController {
    // configuración
    public void setCantidadBots(int cantidad)
    public void initialize()
    public void iniciarJuego()

    // interacciones jugador
    private void seleccionarCarta(int index)
    private void tirarCartaSeleccionada()
    private void bloquearJugador()
    private void desbloquearJugador()

    // flujo bots
    private void jugarBotRecursivo(int indexBot)

    // UI
    private void actualizarInterfaz()
    private void mostrarCartaJugador(int index, ImageView imgView)
    private void mostrarImagen(ImageView imgView, Carta carta)
    private int elegirValorAs()
    private void mostrarAlertaFinDeJuego(String mensaje)
    private void reiniciarJuego()
}

Observaciones: el controlador usa JavaFX (FXML) y clases del paquete model como Juego, Bot y Carta. Maneja estados para evitar reentradas y anima los turnos de bots con PauseTransition.

Bot

Modelo simple para un jugador controlado por IA. Mantiene su mano y elige una carta válida.

package com.cincuentazo.model;

public class Bot {
    // campos
    private String nombre;
    private List mano;

    // constructor
    public Bot(String nombre)

    // getters / setters
    public void setMano(List mano)
    public List getMano()
    public String getNombre()
    public void agregarCarta(Carta c)

    // Lógica de selección IA
    public Carta elegirCarta(int sumaMesa)
}

Nota: el método elegirCarta itera sobre la mano y elimina la carta elegida (mano.remove(c)) antes de retornarla; esto altera la lista durante la iteración en el código fuente original.

Juego

Clase que contiene el mazo, la mesa y el jugador. Controla la lógica principal de jugar cartas y repartir.

package com.cincuentazo.model;

public class Juego {
    private Mazo mazo;
    private Mesa mesa;
    private Jugador jugador;
    private boolean jugadorPerdio;

    public Juego(String nombre)

    // acciones del juego
    public boolean jugarCarta(Carta carta, int valorAs)
    public Carta robarCarta()
    public Mazo getMazo()
    public boolean jugadorPerdio()
    public Jugador getJugador()
    public Mesa getMesa()
}

El método jugarCarta valida que la suma no supere 50, actualiza la mesa, quita la carta de la mano del jugador y roba si el mazo no está vacío.
